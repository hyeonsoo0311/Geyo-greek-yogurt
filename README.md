# 오늘도 출근 완료

음료를 제공해 평범한 한국인 직장인의 HP를 관리하고 30초 동안 퇴근까지 버티는 반응형 픽셀 아트 웹 게임입니다.

## 실행

```powershell
npm run dev
```

브라우저에서 `http://localhost:4173`을 엽니다.

GitHub Pages 배포는 `main` 브랜치에 푸시하면 자동으로 실행됩니다.

## 조작

- 캐릭터 선택 후 즉시 시작
- 게임 중 `캐릭터 변경`으로 다시 선택
- 음료 카드 클릭 또는 키보드 `1`~`4`
- 스페이스바: 일시정지 / 계속하기
- 모든 음료: HP +20
- GEGA COFFEE와 RED BEAR 합산 4잔: 구급차 응급실 엔딩
- 그요 레몬쥬스와 그요 BOSS TEA 합산 4잔: 건강한 퇴근 엔딩
- 카페인 1000mg 이상: 구급차 응급실 엔딩
- 그요 레몬쥬스 3잔: 밝은 표정
- 그요 BOSS TEA 3병: Relax 표정
